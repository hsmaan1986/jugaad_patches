 INTRODUCTION
------------

The Atitude Module module makes a integrations with the Webmeetings API from Atiude.


REQUIREMENTS
------------

This module requires the following modules:

* Views (https://drupal.org/project/views)

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit:
   https://drupal.org/documentation/install/modules-themes/modules-8
   for further information.



CONFIGURATION
-------------

Go to admin/config/atitude_webmeeting/apisettings and set the API Credentials.

HOW TO USE
---------------
* Add Atitude Video field to your content type. 



 * Go to the Block Layout and in the decided region click in place block and choice Atitude Video Block:

   - Use the Content Types visibility form to select which content you want to add this feature.
